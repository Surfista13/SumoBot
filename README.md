# SumoBot
Programme C++ pour robot de combat arduino SumoBot

L'objectif de ce programme est de gérer un robot sumo de maximum 500 grammes et 10cm par 10cm et de le rendre apte à combattre un autre robot du même type sur une surface ronde de 77cm de diamètre
Le robot est equipé de :
  - deux capteurs utltrasons 
  - deux capteurs de lignes situés à l'avant du robot
  - 4 moteurs 6V 75:1
  - une carte arduino nano

Le programme doit gérer le fait de ne pas dépasser les lignes blanches délimitant la sruface de combat et rechercher de manière optimale l'opposant et le pousser en dehors de cette surface de combat.
